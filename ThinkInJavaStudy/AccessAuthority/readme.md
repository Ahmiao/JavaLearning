# 6. 访问控制权限
+ Java一共有四种访问权限，按照权限由大到小分别为public、protected、default和private，如果省略了访问修饰符，那访问权限就是defualt。


| 访问权限 | 含义 | 本类| 本包的类 | 非本类的包 | 非本类的包非子类 |
|---|---|---|---|---|---|
| public | 公共的 |  是| 是 |是  |是  |
|  protected|  保护访问权限| 是 |是  | 是 |否  |
| default |包访问权限  |是  |是  |否 |否  |
|  private|私有的  |是  |否  | 否 |  否|
